Solution

Case Study · Medical Imaging + Deep Learning
Brain Region Segmentation
Sep 2025 – Dec 2025
PythonTensorFlowKeras CNNTransfer LearningOpenCV Dice LossNumPy
Problem
Pixel-level detection of strokes and cerebral oedema in NCCT scans is critical for triage — but severe foreground-background class imbalance and limited annotated data make it extremely challenging.
Solution
Constructed a CNN encoder-decoder with transfer learning for pixel-level lesion detection. Replaced cross-entropy with a custom Dice Loss to resolve class imbalance, surpassing the baseline UNet F1-score on constrained annotated data.
Impact
Validated outputs against radiologist annotations across 200+ scans; precision-recall analysis confirmed near-specialist-level accuracy for stroke triage support.
Metrics
  • 200+ scans validated against radiologist annotations
  • Custom Dice Loss — outperformed baseline UNet F1
  • Near-specialist-level stroke triage accuracy
  • Transfer learning on constrained annotated dataset
